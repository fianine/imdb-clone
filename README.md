# IMDB Clone

## Clone Repo

```
git clone https://github.com/fianine/imdb-clone.git
```
## Install Depedency

```
composer install
```

## Running Migration

```
php artisan migrate
```

## Running Seeder

```
php artisan db:seed
```

## Running Application

```
php artisan serve
```

## Unit Testing

```
vendor/bin/phpunit
```
